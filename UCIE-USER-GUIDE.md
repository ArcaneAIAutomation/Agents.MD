## Getting Started

### Step 1: Access UCIE

1. Go to [https://news.arcane.group](https://news.arcane.group)
2. Log in with your account
3. Click **UCIE** in the navigation menu
4. You'll see the UCIE search interface

### Step 2: Search for a Token

1. Enter a token symbol (e.g., BTC, ETH, SOL)
2. Use autocomplete to find the exact token
3. Click **Analyze** or press Enter
4. Wait 5-15 seconds for comprehensive analysis

The analysis is organized into tabs:
- **Overview** - Executive summary and key findings

- **On-Chain** - Blockchain data and whale activity

## Glossary

**Technical Terms:**

| Term | Definition |
|------|------------|
| **VWAP** | Volume-Weighted Average Price - true market price |
| **RSI** | Relative Strength Index - momentum indicator (0-100) |
| **MACD** | Moving Average Convergence Divergence - trend indicator |
| **EMA** | Exponential Moving Average - smoothed price trend |
| **TVL** | Total Value Locked - assets in DeFi protocol |
| **Gini Coefficient** | Measure of token distribution inequality |
| **Funding Rate** | Cost to hold perpetual futures position |
| **Open Interest** | Total value of open futures contracts |
| **Liquidation** | Forced closure of leveraged position |
