6. Responsive Behavior

Desktop navigation automatically hides on mobile/tablet and shows on desktop.

Breakpoints:

  • Mobile (0-640px): Desktop nav hidden, hamburger menu visible
  • Tablet (641px-1024px): Desktop nav hidden, hamburger menu visible
  • Desktop (1025px+): Desktop nav visible, hamburger menu hidden
This nav is only visible on desktop

Responsive: Resize your browser window to see the navigation hide/show at the 1025px breakpoint.

Implementation Guide

How to use the desktop navigation system in your components. 

<!-- Basic Desktop Nav -->
<nav className="desktop-nav">
  <a href="#" className="nav-link">Link 1</a>
  <a href="#" className="nav-link active">Link 2</a>
  <a href="#" className="nav-link">Link 3</a>
</nav>

<!-- With Layout Variants -->
<nav className="desktop-nav desktop-nav-center">
  <a href="#" className="nav-link">Centered Links</a>
</nav>

<!-- Full Header Example -->
<header>
  <a href="/" className="desktop-nav-brand">
    Logo
  </a>
  <nav className="desktop-nav">
    <a href="#" className="nav-link">Nav Link</a>
  </nav>
  <button className="btn-bitcoin-primary">CTA</button>
</header>

CSS Classes: All styles are in styles/globals.css. No JavaScript required for basic functionality.
